leak泄露

John Doe
2025-07-22 (Updated: 2025-07-22)

之前写的leak泄露，整理了一下搬到这

特征：
一般来说 leak泄露类型的题目和p，q之间有直接的关系，那么剩下就是靠数学推导

前置需要：同余的性质和运算，费马小定理，欧拉定理

1.leak = （p^q + q^p)%n

定义一个变量leak leak = (pq + qp) (mod  n)

从上面的定义可以推出： leak ≡ pq + qp (mod  n)

另外，题目中也给出了： leak ≡ p (mod  q) 同理： leak ≡ q (mod  p)

因此，我们可以写出： leak = p + k1q = q + k2p 其中k1和k2是某个整数。

重新整理上述等式： (k1 − 1)q = (k2 − 1)p

考虑两种情况： 第一种情况 k1 = k3p + 1 k2 = k4q + 1 这种情况下，由于leak < n，所以不成立。 第二种情况 k1 = 1 k2 = 1 将k1=1代回leak = p + k1*q的等式中： leak = p + (1)q 因此得出结论： leak = p + q

2.leak = d+p+q

leak = d + p + q ϕn = n − (p + q) + 1 h = d + (p + q)e d = 1 + kϕn ed − 1 = k(n + 1 − h) + kd (e − k)d = k(n + 1 − h) + 1 [k(n + 1 − h) + 1] (mod  e − k) = 0遍历k即可

当e比较小的时候，k ≈e*leak /n

3.leak1=(p+q)e % n, leak2=(p+e)q % n

leak2 = (p + e)q  = (p + e) (mod  q) p = leak2 − e (mod  q)

leak1 = (p + q)e  = pe  = (leak2 − e)e (mod  q)

所以有 leak1 − (leak2 − e)e = k ⋅ q q = gcd (k ⋅ q, n)

kq = leak1 − (leak2 − e)e q = gcd (kq, n)
