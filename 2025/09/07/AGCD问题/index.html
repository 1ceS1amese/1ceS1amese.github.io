AGCD 问题

如果说我们知道 x1, …, xl 满足 x1 = pq1，x2 = pq2，…xi = pqi 那么我们可以发现各个数直接有一个最大公约数p，然后用gcd(x1, x2…xi)然后就可以分解各个数 但如果此时我们加入误差e1, e2​将式子构造为形如 x1 = pq1 + e1, x2 = pq2 + e2 那么此时我们就没办法通过求公约数来解决这个问题了 在e1和e2较小的情况下我们可以通过爆破e1, e2方式来求解gcd(x1 − e1, x2 − e2)，但是在$e_1和e_2较大时我们是无法解决的，这时就是需要用其他的解决方法了
