# >_ 1ceS1amese

A terminal-style personal hub page built with React + Ant Design.  
Interact with a virtual filesystem, run commands, and navigate to personal pages.

## Stack

- React 18 + TypeScript
- Ant Design 5 · Illustration theme
- Vite 5 · HashRouter · GitHub Pages

## Quick Start

```bash
npm install
npm run dev    # dev server
npm run build  # output → dist/
```
